# 🛒 성인 인증 키오스크 (Self-Check Kiosk v2.0)

## 📖 프로젝트 소개
기존 **Self-Check Kiosk(v1.0)** 프로젝트를 고도화하여 **관리자 보안 및 로그 관리 기능**을 추가한 개인 프로젝트입니다.
무인 매장 키오스크 환경을 가정하여, 주류 등 성인 인증이 필요한 상품 구매 시 발생하는 법적 책임 동의 및 인증 로그를 기록하고, 관리자만이 안전하게 해당 로그를 열람할 수 있도록 시스템을 구축했습니다.

> **v2.0 주요 변경 사항:**
> * 기존 팀 프로젝트의 핵심 로직(바코드 인식, JSON 저장) 계승
> * **FastAPI** 기반의 백엔드 API 서버 구축
> * **관리자 인증 시스템** 도입 (Bcrypt 비밀번호 암호화)
> * **관리자 전용 로그 뷰어** 구현 (Swagger UI 활용)

---

## 🚀 주요 기능

### 1. 사용자 (User) - 키오스크 모드
* **바코드 스캔 시뮬레이션:** 상품 바코드 인식 시 성인 인증 절차 자동 실행
* **3단계 인증 절차:** 성인 인증 및 법적 책임 동의 절차 수행
* **로그 자동 기록:** 인증 결과, OCR 데이터(시뮬레이션), 동의 내역이 `logs.json`에 암호화/저장

### 2. 관리자 (Admin) - 관리 모드 (New ✨)
* **보안 로그인:** `admins.json` 기반의 ID/PW 검증 (Bcrypt 해싱 적용)
* **접근 제어:** 로그인하지 않은 사용자는 로그 데이터 접근 불가 (401 Unauthorized)
* **로그 열람:** 관리자 권한 승인 시에만 `logs.json` 데이터 조회 가능
